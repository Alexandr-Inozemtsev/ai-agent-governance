# Project Onboarding Guide

1. Выбрать domain policy.
2. Создать project policy из `project-overrides/project-policy-template.yaml`.
3. Указать repository.
4. Указать Trello board.
5. Указать allowed agents и disabled agents.
6. Указать project-specific restrictions.
7. Настроить bug rules.
8. Запустить quality gate.
9. Начать работу через preflight и owner routing.
